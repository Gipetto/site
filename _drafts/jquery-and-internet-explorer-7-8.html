Having fun debugging IE today.

jQuery.trim()

This is a joke, right?

_val = $('.my-input').val();
_val = _val.trim(); // FAIL
// or
_val = $('.my-input').val().trim(); // FAIL
// but
_val = $.trim($('.my-input').val()); // works

The only way I could trim the value was to use $.trim() explicitly. Ugh.

Url Hashes

Another fun one: The URL hash. Calling jQuery.attr('href') SHOULD return me the raw attribute value, right? Wrong, but only in IE.

The HTML:
<a class="tab-link" href="#my-hash">url</a>

$('.tab-link').click(function() {
hash = $(this).attr('href'); // http://my-site.com/#my-hash - FAIL
// so
hash = $(this).attr(_this.attr('href').slice(_this.attr('href').indexOf('#'))); // #my-hash - works
});

Ugh.
